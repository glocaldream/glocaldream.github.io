안녕하세요 여러분 ~! 😎 오늘은 github page를 만드는 법에 대해서 간단하게 알아볼게요 !

Github page 란?

GitHub의 Pages란(이하 페이지) GitHub에서 무료로 호스팅하는 공개 웹페이지이다. GitHub에서 제공하는 페이지 제작 도구로 쉽게 페이지를 만들어 공개할 수도 있고, 로컬에서 git을 이용해 수동으로 만들 수도 있다.

Github page 장점

  • github에서 호스팅을 해준다.
  • 간단한 웹사이트에 최적이다.
  • HTML/ mark down 을 사용할 수 있다.

자 이제 만들어 보겠습니다!

1. Github에 로그인 후 Repositores에 새로운 레퍼지토리를 생성해준다.

2. Repository name에 userid.github.io를 입력후 Create repository클릭

* 다른이름으로 설정하게 될 경우에는 https://userid.github.io/[다른이름].github.io 이렇게 설정되오니 주의해주세요


3. 헤더에 Setting > GitHub Pages > Source > master branch 로 설정합니다

이제 테마를 설정할건데요. github page에서 추천해주는 도구 Jekyll theme 를 사용해 보겠습니다.

여기서 지킬테마가 무엇인지 알아보도록 하겠습니다!


Jekyll theme ?

지킬(Jekyll)은 정적 웹사이트 빌더(static site builder) 도구입니다.

장점
  • 사용법이 간단하다
  • 마크다운(Markdown) 기반으로 되어있다.
  • GitHub Pages에서 무료 호스팅 지원
  • 다양한 테마 및 플러그인, 편집기(editor) 생태계 보유
  • 확장 가능한 테마 시스템

4. Jekyll theme 페이지에서 맘에드는 테마를 고릅니다.

저는 Type on Strap 라는 테마로 결정했습니다!


5. 해당하는 테마의 깃허브 홈페이지에 들어가서 clone or download -> Download.zip 클릭 합니다.

저는 필요한 파일만 가져갈려고 download.zip 클릭했어요 !

내 id로 된 레퍼지토리를 문서에 clone 해줍니다!
